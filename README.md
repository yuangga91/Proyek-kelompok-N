# MENU MAKAN HEMAT

**Solusi Pengelolaan Makanan Hemat untuk Anak Kos Mahasiswa Undip Semarang**

## Deskripsi Proyek

MENU MAKAN HEMAT adalah aplikasi web prototype yang dirancang khusus untuk membantu mahasiswa, terutama anak kos di Universitas Diponegoro Semarang, mengatur pola makan hemat, sehat, dan sesuai kebutuhan gizi dengan budget terbatas.

## Fitur Utama

### 1. ✅ Sistem Penentuan Menu Termurah Mingguan
- Rekomendasi menu makanan selama 1 minggu
- Penyesuaian menu berdasarkan budget pengguna
- Tampilan total pengeluaran harian dan mingguan
- Optimasi menu yang hemat namun bergizi

### 2. ✅ Rekomendasi Tempat Makan dan Pesan Antar
- Rekomendasi warung makan murah sekitar kampus
- Estimasi harga makanan
- Opsi makanan paling ekonomis

### 3. ✅ Kalkulator Kebutuhan Kalori Harian
- Penghitung kalori berdasarkan jenis kelamin, usia, tinggi badan, berat badan
- Estimasi kebutuhan kalori harian
- Rekomendasi jumlah makan per hari
- Saran pola makan sesuai kebutuhan tubuh

### 4. ✅ Pilihan Jenis Pola Makan
- Makanan hemat sehat
- Makanan hemat tidak sehat (fast food/junk food hemat)
- Personalisasi berdasarkan preferensi pengguna

### 5. ✅ Filter Preferensi Nutrisi
- Tinggi serat
- Tinggi protein
- Tinggi lemak
- Seimbang

### 6. ✅ Filter Alergi Makanan
- Seafood
- Telur
- Susu
- Kacang
- Gluten
- Dan lainnya

### 7. ✅ Rekomendasi Budget Bahan Masakan
- Rekomendasi bahan masakan murah
- Estimasi total biaya memasak
- Panduan untuk yang lebih suka memasak sendiri

### 8. ✅ Rekomendasi Resep Sesuai Budget
- Daftar resep sederhana
- Langkah memasak singkat
- Estimasi biaya per resep
- Bahan yang diperlukan
- Estimasi kalori

### 9. ✅ Pengingat Waktu Makan
- Notifikasi sarapan
- Notifikasi makan siang
- Notifikasi makan malam
- Notifikasi browser dan popup sederhana

## Teknologi yang Digunakan

- **HTML5** - Struktur dan semantic markup
- **CSS3** - Styling responsif, modern, dan user-friendly
- **JavaScript (ES6+)** - Interaksi dinamis dan logika aplikasi
- **LocalStorage** - Penyimpanan data pengguna lokal

## Desain Visual

- **Tema Warna**: Dominan hijau dan oranye (identik dengan makanan sehat dan hemat)
- **Gaya**: Minimalis, modern, clean UI
- **Responsif**: Kompatibel dengan desktop dan mobile

## Cara Menggunakan

1. Buka file `index.html` di browser
2. Isi data pribadi (jenis kelamin, usia, tinggi, berat, budget)
3. Pilih preferensi makan dan alergi
4. Sistem akan merekomendasikan menu mingguan
5. Lihat detail budget, kalori, dan resep
6. Aktifkan reminder makan
